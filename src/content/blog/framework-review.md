I've been using the Framework 16 as my daily driver for six months now.

## Build Quality

It's solid. Not MacBook-level premium, but it feels well-made. The modular input deck is cool — I have the numpad module on the right and a spacer on the left.

## Performance

The Ryzen AI 9 HX 370 is a beast. Compiles are fast, Docker containers spin up instantly, and the integrated Radeon 890M handles light gaming fine.

## The Modular Thing

Being able to swap ports, upgrade RAM, and replace the battery is genuinely useful. I already upgraded the SSD from 500GB to 1TB by just popping in a new drive.

## Linux Support

Bluefin (Fedora) runs flawlessly. Suspend, Bluetooth, WiFi, external displays — everything works out of the box. Framework clearly tests on Linux.
