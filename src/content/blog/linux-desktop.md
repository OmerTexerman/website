I got a Framework 16 and installed Bluefin (a Fedora-based immutable distro). It's been great.

## The Good

Everything just works. Bluetooth, WiFi, suspend, external displays. GNOME 49 is solid. The Framework hardware is excellent — I can swap modules, upgrade RAM, replace the battery.

## The Adjustment

Coming from macOS, the biggest change was the software ecosystem. But between Flatpak and the terminal, I haven't missed much. VS Code (well, Cursor) runs perfectly. Docker is native. No Homebrew needed.

## Would I Go Back?

No. The repairability alone is worth it. And Linux on modern hardware is genuinely good now.
